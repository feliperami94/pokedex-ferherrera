# Execute in development

1. Clone the repository
2. Execute

```
npm i 
```
3. Make sure you have the Nest CLI installed
```
npm i -g @nestjs/cli
```

4. Run the database container
```
docker-compose up -d
```

## Stack used
* MongoDB
* Nest
